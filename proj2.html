Butler Robot

About The project

In this project, we implement the YOLO algorithm for realtime obstacle detection, leveraging its high accuracy and processing speed. YOLO treats object detection as a regression problem, predicting bounding boxes and class probabilities in a single pass over the input image. The practical implications of this project are significant, as it enables the creation of custom object detection and recognition systems tailored to specific application domains. The Butler Robot detects household items on the floor and informs the resident through email notification. This is useful for people who cannot get up and find things on a regular basis due to health issues. This also detects the owner and if found on floor, sends out an email to closest family relative.
